POS System Website
================

This is a beautiful POS system website built with HTML, CSS, and JavaScript.

Getting Started
---------------

1. Open `index.html` in your web browser.
2. Explore the website and add products to your cart.

Features
--------

* Responsive design
* Modern UI
* Product catalog
* Cart system
